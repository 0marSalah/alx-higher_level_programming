
===========================
How to Use 1-my_list.py
===========================

This module defines a class `MyList` that inherits from `list`.

Usage
=====
>>> my_list = __import__('1-my_list').MyList()
>>> my_list.append(1)
>>> my_list
[1]
>>> my_list.append(2)
>>> my_list
[1, 2]
>>> my_list.append(3)
>>> my_list
[1, 2, 3]
>>> my_list.append(4)
>>> my_list
[1, 2, 3, 4]

>>> my_list.print_sorted()
[1, 2, 3, 4]

>>> my_list.pop()
4
>>> my_list
[1, 2, 3]
>>> my_list.pop()
3
